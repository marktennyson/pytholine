Given a string s containing only lowercase letters, implement a function most_frequent_k_chars(s: str, k: int) -> List[str] that returns a list of the k most frequent characters in s, in decreasing order of frequency. If there are ties in frequency, the characters should be sorted in lexicographically increasing order. For example, if s is "aabbbcccd", and k is 2, the function should return ['b', 'c'], since b and c both occur 3 times (which is the maximum frequency), and b comes before c in lexicographic order. Note that the input string s can have length up to 10^4, and the value of k can be up to the length of s. You can assume that the input is valid and that k is always less than or equal to the number of distinct characters in s.
